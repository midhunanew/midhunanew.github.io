Find and Replace console.log using reg.ex search!!!

Tested with WebStorm Only, must be useful any tool supporting reg-ex find and replace.

find console

console.log\(.*\)

comment console with '//'

console.log\((.*)\) //console.log\($1\)

use this to remove commented console.logs ('//') in any regx finder

//\s+console.log\(.*\)

var DEBUG = false;

then add this so console can be controlled globally ;)

remove incuding new line

\s+ console.log\((.*)\)?; if (DEBUG) console.log\($1\);

remove without newline

[^\S\n] console.log\((.*)\)?; if (DEBUG) console.log\($1\;

no worries, if you want remove just replace with empty string.

[^\S\n] console.log\((.*)\)?;
